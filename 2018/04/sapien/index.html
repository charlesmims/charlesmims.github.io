Sapien

Tue Apr 24, 2018 by Charles Mims

The other night my wife and I were talking about an idea I had for a blockchain-based social media site that took the best things about reddit, but ran distributed on users’ systems via blockchain, where people could buy credits and donate fractions of credits to content creators, and communities could form about geographies and interests, using the platform for news, discussion, community engagement, and just staying in touch with old friends. It seemed like a good idea at the time, and as it happens I wasn’t the first person to come up with it.
I came across https://sapien.network today. It looks interesting, touches several of the points above. It’s still in closed beta, to which you can join a waitlist, so it remains to be seen how it will work in practice. I’ll write more about it as I find out more.
I did notice they’re using telegram for communication about the project, and using it well.
If you know anything about this project or similar community-centric alternatives to Facebook, let me know via telegram.
