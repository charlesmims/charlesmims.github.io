Calling an API with go

Thu Jan 19, 2017 by Charles Mims

I’ve been trying to learn go (golang) over the past couple months. The documentation at golang.org is fantastic and only takes about a half a day to get through, but I’m still working on getting really comfortable thinking in go. Coming from python, I find the language beautiful, but not quite natural. I’m getting there.

I wrote this little program to fetch the latest bitcoin price from the coinbase API and print it to stdout.

package main

import (
  "encoding/json"
  "fmt"
  "io/ioutil"
  "log"
  "net/http"
)

type (
  Notice struct {
    ID      string `json:"id"`
    Message string `json:"message"`
    URL     string `json:"url,omitempty"`
  }

  Amount struct {
    Data struct {
      Amount   string `json:"amount"`
      Currency string `json:"currency"`
    } `json:"data"`
    Warnings []*Notice `json:"warnings,omitempty"`
  }
)

func main() {
  client := &http.Client{}
  req, err := http.NewRequest("GET", "https://api.coinbase.com/v2/prices/spot?currency=USD", nil)
  if err != nil {
    log.Fatal(err)
  }
  req.Header.Set("CB-VERSION","2017-06-21")
  resp, err := client.Do(req)
  if err != nil {
    log.Fatal(err)
  }
  defer resp.Body.Close()
  data, err := ioutil.ReadAll(resp.Body)
  resp.Body.Close()
  if err != nil {
    log.Fatal(err)
  }
  var dec Amount
  json.Unmarshal(data, &dec)
  fmt.Printf("%s\n", dec.Data.Amount)
}

To build this, I ran go install in the directory where I have this saved as main.go (~/go/src/github.com/charlesmims/btc). Now I can run btc and get the current bitcoin price.

cmims@MBP:~$ btc
964.15
